# pychalk Cookbook

Welcome to the official pychalk Cookbook. This repository contains practical examples, code snippets, and guidelines for using axon square's **pychalk** (python wrapper for the CHALK - Channeling Analysis Learning Kernel - library).

**Please Note:** pychalk is a commercial, paid software library. A valid commercial license is required to run the examples and utilize the underlying engine provided in this repository.

pychalk requires Nvidia SDK version 12.8 to run.
cookbook is based on axon-square-pychalk-0.1.1

## Documentation

The complete and up-to-date API documentation is automatically generated and hosted via GitHub Pages. 
You can consult it here: [PyChalk API Documentation](https://sebastien-dubail.github.io/pychalk-cookbook/)
